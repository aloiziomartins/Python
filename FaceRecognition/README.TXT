Aplicação para cadastro de usuário e captura de imagens do usuário para treinar e fazer reconhecimento facial usando biblioteca opencv-ṕyhton.
Após criar o ambiente virtual "virtual env" e instalar as bibliotecas necessários disponíveis em requierments.txt inicar o arquivo main.py, para criar o banco dados e
iniciar o menu com opções de cadastro, treinar e reconhecer.
A cada novo cadastro de usuário o arquivo "classificadorLBPH_V1.yml" será atualizado com o reconhecimento fácial de todos os usuários, a função treinar() é para possibilidade de no
futuro adicionar usuário e apontar diretórios diretórios que contenha fotos e não necessáriamente serem feitos pela Webcam.

Ambiente que aplicação foi usado:
OS: Ubuntu 22.04
